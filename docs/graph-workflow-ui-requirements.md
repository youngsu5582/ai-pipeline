# Graph Workflow UI 요구사항

## Original Requirement
"현재 카드 기반 대시보드를 그래프 노드 스타일의 Visual Workflow Builder로 변경하고 싶음. 상단에 시간 타임라인, 드래그로 관리, 작업 간 의존성 연결 기능 원함."

---

## Clarified Requirements

### Goal
카드 기반 작업 목록을 그래프 노드 기반 Visual Workflow UI로 전환

### Scope

#### Phase 1 (1차 구현)
- [ ] 노드 기반 작업 표시 (기존 카드 → 노드)
- [ ] 드래그로 노드 위치 이동
- [ ] 드래그로 노드 간 연결선 생성
- [ ] 기존 기능 100% 유지 (실행/활성화/비활성화, cron 편집, 카테고리 필터)

#### Phase 2 (2차 이후 확장)
- [ ] 상단 타임라인 (일/주/월 전환 가능, 스케줄 시각화 용도)
- [ ] 연결선 기반 순차/조건부 실행 (성공/실패 분기)
- [ ] 캔버스 클릭 + 툴바 버튼으로 새 작업 추가
- [ ] 타임라인에서 cron 스케줄 시각화

### Constraints
- 기존 기능 전체 유지
- 노드는 각각 독립 실행 가능해야 함
- 연결선은 선택적 (없어도 개별 실행 가능)

### Success Criteria (Phase 1)
- [x] 기존 모든 작업이 노드로 표시됨
- [x] 노드를 드래그해서 위치 변경 가능
- [x] 노드 간 연결선 그리기 가능
- [x] 기존 실행/활성화/비활성화/cron 편집 기능 동작

---

## Decisions Made

| Question | Decision |
|----------|----------|
| 타임라인 용도 | 스케줄 시각화 (Phase 2) |
| 노드 연결 의미 | 개별 실행 + 조건부 실행 둘 다 (Phase 2) |
| 조건 기준 | 성공/실패 분기 + 완료 후 실행 둘 다 |
| 기존 기능 유지 | 전체 유지 |
| 드래그 범위 | 위치 이동 + 연결선 그리기 |
| 타임라인 단위 | 일/주/월 전환 가능 (Phase 2) |
| 작업 추가 | 캔버스 클릭 + 툴바 버튼 (Phase 2) |
| 1차 구현 범위 | 기본 그래프 UI |
